字符型：char,signed char(0-255),unsigned char(-128-127),具体运行时char只使用两种类型，由编译器决定

变量：

声明和定义的关系：变量的定义必须出现在且只能出现在一个文件中， 其他用到该变量的文件必须对其进行声明，不可重复定义。

复合类型：引用和指针

左值引用：引用类型引用另外一种类型，一旦初始化完成，引用将和它的初始值对象一直绑定在一起。

引用并非对象，他只是为一个已经存在的对象所起的另一个名字。对于引用做的操作，实际上是对引用的对象做操作

指针：

存放某个对象的地址，不能指向引用

指针本身就是一个对象，允许对指针赋值和拷贝，在指针的生命周期中可以先后指向几个不同的对象；指针无需在定义时赋初值。

取地址符：&

指针类型和对象类型要相匹配。

如果指针指向了一个对象（有效指针），则允许使用解引用符*来访问该对象

多级指针：

int *q 可以把它分为两部分看，即 int 和 (q)，后面 (q) 中的“”表示 q 是一个指针变量，前面的 int 表示指针变量 q 只能存放 int* 型变量的地址。对于二级指针甚至多级指针，我们都可以把它拆成两部分。

首先不管是多少级的指针变量，它都是一个指针变量，指针变量就是一个“”，其余的“”表示的是这个指针变量只能存放什么类型变量的地址。比如“int*a；”表示指针变量 a 只能存放 int 型变量的地址。

也可定义指向常量的指针，用于保存地址，但不能用于改变其所指对象的值，

常量表达式：值不会改变且在编译过程中就能得到计算结果的表达式，

预处理器：

我们在编写程序时，同时包含一个头文件多次是一件常见的事情，当然同时多次包含一个头文件可能会引起不必要的错误。

使得头文件安全的通用做法，是使用预处理器定义头文件保护符。头文件保护符用于避免在已经见到头文件的情况下重新处理该头文件的内容。
